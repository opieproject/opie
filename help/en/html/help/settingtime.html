Setting the Time

To set the current time:
- Tap the up/down arrows on the Hour box to set the hour.
- Tap the up/down arrows on the Minute box to set the minutes.
- Tap the AM/PM dropdown list to select the time of day.

To change the time format:
- Tap the Time Format dropdown list
- Select either a 12 hour or 24 hour time format

Note: When the 24 hour time format is selected the AM/PM dropdown list becomes unavailable.

See also Setting the Date.
