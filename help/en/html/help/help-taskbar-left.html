Left of the Taskbar

The Qt menu appears at the far left of the taskbar.

To start a program using the taskbar;
- Tap the OPIE icon
- Then Applications, Games, Settings or Documents
- Additionally installed plugins appear .
- Then the application or document you want to use.

To the right of this menu is the Input Methods icon. Tap this to pop up a menu of input methods, then tap the input method you want to use.
